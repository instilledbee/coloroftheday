About

Add ?mm/dd/yyyy to the end of the URL to send a specific date to the script, e.g. http://instilledbee.net/projects/coloroftheday?11/27/2009

Color of the Day is a simple JS project by InstilledBee (Junvic), that involves providing a unique hexadecimal color value given a date, effectively providing a hash of sorts. The algorithm is written from scratch, and parts of the code rely on jQuery (and the glDatePicker plugin) for assistance.

Fork this on GitHub, or contact me for comments/suggestions. Cheers!
